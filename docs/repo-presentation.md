## Public Talking Point

Position the repo as a practical engineering project that organizes local tabular ML workflows into one UI + CLI codebase, with honest boundaries around what is and is not implemented.

Lead with the implemented local flow: validation, profiling, benchmark, experiment, and prediction. Do not feature notebook mode or `colab_mcp` as finished capabilities in public copy.
